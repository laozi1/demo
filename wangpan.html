基于 Ubuntu + nextCloud 搭建自己的私人网盘

基础设置

任务时间：时间未知

切换为 root 账号

Ubutu 系统默认登录的用户为非 root 权限用户，为了能正常安装 nextCloud，需要切换为 root 账号。执行如下命令即可：

sudo su

更新及安装基础包

请依次运行如下命令，有遇到询问的Is this ok [y/d/N]的时候直接键入y并回车即可。

apt-get update
apt-get install snap
apt-get install snapd

安装 nextCloud

任务时间：时间未知

进入安装

运行如下命令：

snap install nextcloud

国内网络下载有点慢，需要耐心等待。笔者差不多1h 才安装好，建议可以睡个小觉再回来看进度。

看到如下图则证明安装成功。

设置 nextCloud

任务时间：时间未知

创建用户

访问本机ip (http://<您的 CVM IP 地址>/) 即可进入 nextCloud 的安装页面。

根据个人需求输入用户名创建用户即可。

管理

创建成功即进入管理界面，与一般的网盘使用上无多大差别，也可以按照提示下载使用移动端 app 管理文件。
